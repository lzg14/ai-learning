# AGENTS.md

This file provides instructions and context for AI agents working with code in this repository.

## 仓库性质

这是一个 **AI Agent 学习课程** 的讲义仓库，包含 40 个学习日（Day01–Day40）+ 4 个扩展阅读的课程内容，以 Markdown 文件形式保存。

**不是代码项目**，没有可执行的代码、构建工具或测试套件。

## 内容结构

- **Day01–Day40**：40 天完整学习路径，覆盖 AI Agent 全栈知识
- 课程分为 3 个阶段：
  - 筑基阶段（Day 01–15）：基础概念、函数调用、Agent 框架
  - 核心阶段（Day 16–30）：Agent 核心能力、记忆、规划、协作
  - 进阶阶段（Day 31–40）：高级技术、微调、安全、生产化
- **Ext01–Ext04**：扩展阅读（哲学与社会思考）

## 使用说明

- 本仓库仅用于内容查阅与学习，不要尝试"构建"或"运行"任何东西
- 涉及修改时，只修改 Markdown 文件内容，不改动文件编码（保持 UTF-8）
