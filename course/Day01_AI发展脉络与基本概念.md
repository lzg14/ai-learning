## Part 1：AI 发展简史

### 1956-1980：人工智能的诞生与第一次寒冬

**达特茅斯会议（1956）**
- 6位科学家（麦卡锡、明斯基、香农等）在达特茅斯学院召开会议
- 正式提出"人工智能"概念
- 目标：让机器像人一样思考、推理、学习

**早期成果**
- ELIZA（1966）：第一个聊天机器人，用规则模拟心理医生
  - 原理：模式匹配 + 固定回复，不是真正的理解
  - 意义：证明机器可以模拟对话
- 专家系统（1970s）：把人类专家知识编码成规则
  - 代表：MYCIN（诊断细菌感染）、DENDRAL（分析化学结构）
  - 局限：只能处理规则明确的领域，无法泛化

**第一次寒冬（1970s）**
- 期望过高但技术不成熟
- 计算能力不足，无法实现复杂推理
- 资金减少，研究停滞

---

### 1980-1995：机器学习的兴起

**反向传播算法（1986）**
- Rumelhart、Hinton 等人提出
- 让多层神经网络真正能够学习
- 深度学习的大门正式打开

**统计学习崛起**
- SVM（支持向量机）：核方法让机器学习效果大幅提升
- 决策树、随机森林：可解释性强的机器学习方法
- 语音识别、图像分类开始使用机器学习

**关键事件**
- 1997年：IBM 深蓝击败国际象棋冠军卡斯帕罗夫
  - 这不是AI的"思考"，而是暴力搜索 + 棋局评估
  - 在规则明确、搜索空间可穷举的领域，AI超越人类

---

### 2012-至今：深度学习革命

**AlexNet 时刻（2012）**

```
ImageNet 图像识别挑战赛：
2011年冠军错误率：25%
2012年 AlexNet（CNN）：16%  ——  深度学习震惊世界
2015年 ResNet：3.6%  ——  超越人类水平（5%）
```

**深度学习的三大支柱**
1. **海量数据**：互联网时代，数据唾手可得
2. **强大算力**：GPU/TPU 让训练大规模神经网络成为可能
3. **算法突破**：ReLU激活、Batch Normalization、ResNet等

## Part 2：LLM 时代的到来与大模型崛起

### Transformer 革命

**注意力机制的核心思想**

```python
# 传统RNN：逐字处理，难以处理长距离依赖
"我爱[吃]...苹果"  →  RNN需要从"爱"记住到"苹果"

# Transformer：并行处理，用注意力机制直接关联任意位置
"我爱[吃]...苹果"  →  注意力机制直接连接"爱"和"苹果"
```

**为什么 Transformer 如此重要？**
1. **并行计算**：可以同时处理整个序列，训练速度大幅提升
2. **长距离依赖**：通过注意力机制直接建立任意位置的联系
3. **可扩展性**：模型越大，效果越好，没有明显上限

微调阶段：

### 逐个解析

**AI（人工智能）**
- 最广泛的概念，1956年正式提出
- 目标：让机器像人一样思考和解决问题
- 包含：规则系统、机器学习、深度学习等所有方法

**ML（机器学习）**
- 核心理念：不手工写规则，让机器从数据中学习
- 代表方法：线性回归、SVM、决策树、随机森林、神经网络
- 突破：1990年代在语音识别、图像分类上超越传统方法

**DL（深度学习）**
- 使用多层神经网络（现在通常几十到上百层）
- 自动学习特征表示，无需手工设计特征
- 2012年后成为AI领域主流方法

**LLM（大语言模型）**
- Transformer架构 + 超大规模参数
- 预训练在海量文本上，学习语言规律和世界知识
- 核心能力：文本生成、理解和推理

**Agent（智能体）**
- LLM + 工具 + 规划 + 记忆
- 能自主决策、执行多步骤任务
- 是LLM在实际应用中的形态

---

### LLM 通俗理解：超级阅读者

**把LLM想象成一个"读完了互联网上所有文本的超级读者"**

关键特点：
1. **它不"理解"文字的含义** —— 学的是文字之间的统计规律
2. **它靠概率工作** —— 预测下一个词最可能是什么
3. **它的知识在训练时固定** —— 不知道新发生的事情

**概率预测示例**

```
输入："中国的首都是"
输出：
  "北京" → 概率 98.2%
  "上海" → 概率 0.8%
  "南京" → 概率 0.5%

模型选择概率最高的"北京"
```

**涌现能力**

当模型大到一定程度，会出现意想不到的能力：
- 小模型：不会做数学题
- 大模型：突然会了
- 原因：规模增大时，模型从"记忆"模式进入"推理"模式

## Part 4：动手实践——Python 环境搭建

### 安装 Python

**方式一：官网下载**
1. 打开 https://www.python.org/downloads/
2. 下载 Python 3.10 或更高版本（推荐 3.11/3.12）
3. 运行安装程序

> ⚠️ Windows 用户记得勾选 **"Add Python to PATH"**

**方式二：国内镜像站（更快）**
- 华为云：https://mirrors.huaweicloud.com/python/
- 腾讯云：https://mirrors.cloud.tencent.com/pypi/
- 阿里云：https://mirrors.aliyun.com/pypi/

### 验证安装

```bash
python --version
# 应该显示 Python 3.11.x 或更高版本

pip --version
```

### 安装 uv（比 pip 快 10 倍的包管理器）

```bash
pip install uv
```

### 创建第一个项目

```bash
# 创建项目目录
mkdir llm-learning && cd llm-learning

# 创建虚拟环境
uv venv

# 激活虚拟环境
# Mac/Linux:
source .venv/bin/activate
# Windows:
.venv\Scripts\activate

# 安装常用库
uv pip install requests python-dotenv
```
