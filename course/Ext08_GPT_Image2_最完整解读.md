# Ext08：GPT Image 2 最完整解读——第一个会"思考"的图像模型

> OpenAI的ChatGPT Images 2.0震撼发布，其"思考模式"与多语言支持能力彻底颠覆了图像生成的范式。本文基于深度实测，全面解析其核心突破、应用场景与潜在局限。

---

## 一、哪里能用 · 价格 · 权限

**三端同时全量上线**：ChatGPT、Codex、API。带思考模式（Thinking）的进阶版本，仅ChatGPT Plus、Pro、Business用户能用。

API模型名：**gpt-image-2**，通过Image API（generations/edits）和Responses API（image_generation工具）都能调用。Codex内置图像生成，不需要单独申请API key。

**价格涨幅显著**：high档方图从$0.133涨到$0.211（+59%）；medium档从$0.034涨到$0.053（+56%）。

**尺寸约束**：最大边长≤3840px，长短边比≤3:1，总像素65万~829万，每边都是16px倍数。常用2K方图、2K长方图、4K横竖屏都能跑，2K以上当前是beta阶段。

**调用示例**：
```python
from openai import OpenAI
client = OpenAI(api_key="sk-...")
result = client.images.generate(
    model="gpt-image-2",
    prompt="A horizontal magazine cover, ...",
    size="1920×816",
    quality="high",
)
img_bytes = base64.b64decode(result.data[0].b64_json)
open("cover.png", "wb").write(img_bytes)
```
封面整图分两次画（左1024×1024 + 右1920×816），加起来不到三毛钱人民币。

## 二、第一个会思考的图像模型

**这是这次发布最大的范式变化。**

ChatGPT Images 2.0是OpenAI第一个带思考能力的图像模型。在ChatGPT里选thinking或pro模型时触发，做三件事：
1. 联网搜索实时信息
2. 一次产出最多8张连贯图
3. 自我检查输出质量

**五个代表演示**：
- **演示一**：扒OpenAI官网当前在售的merch做产品海报——模型实时去网站搜真实商品
- **演示二**：在35mm黑板照片上证明"奇数之和等于平方数"——数学可视化推理
- **演示三**：一次画完四页连贯漫画——主角形象在四页里保持稳定，漫画工作流第一次跑通
- **演示四**：抹茶店在不同社交平台的多尺寸广告——一次出Twitter、IG故事、IG信息流、LinkedIn四个尺寸
- **演示五**：基于上传的论文PDF做学术海报——模型读完整篇PDF，提取关键图表，按学术版式输出

> **思考模式的核心价值不是"画得更好"，是"替你想清楚"。** idea到成品之间那段繁琐的脑力活，模型自己接下了。

## 三、文字渲染：从英文跨到非拉丁文字

这是普通用户最直接能感受到的变化。

之前的图像模型在英文和拉丁字母语言上一直比较稳，日文、中文、韩文、印地文、孟加拉文一旦密度上来就崩。ChatGPT Images 2.0在这五种文字上有显著提升：

- **日文**：全篇日文连环漫画，整张图当物理印刷的漫画书页处理
- **印地文**：九种印度语言的书封陈列，所有文字清晰可读
- **中文**：每个汉字都准，包括底部那段超小字号的测试文字

> **中文不再是图像模型的二等公民。这是这一代国内用户最该关心的变化。**

## 四、指令服从和细节渲染

在"按你说的精确去做"这件事上提升明显，特别是：小字、图标、UI元素、密集排版、微妙的风格约束。API端最高支持2K分辨率。

亮点案例：
- 狼的科学杂志编辑页，光面、流畅、排版克制的科学杂志风
- 手写棒球史，笔画粗细带人为不均，右上角一块淡淡的咖啡渍
- 米堆找字——上千粒米中有一粒刻着"GPT Image 2"，跟其他米粒一样大，远看完全找不到

## 五、宽高比拉到3:1到1:3

之前图像模型一律以方图为主，这次从3:1超宽到1:3超高都能跑：
- 3:1超宽篮球扣篮的连环动作分解
- iPhone全景法国南部夏日
- Art Deco风格书签——完整带尺寸标注，含出血、裁切、安全边距，可以直接拿去印刷
- 1:3超高童书风路线图

---

## 六、还做不到的事

- 需要完整、连贯的物理世界模型的任务（折纸指南、魔方拼图）依然吃力
- 极密、极重复的视觉细节（沙粒级别）会逼到模型上限
- 带精确箭头和零件标签的标注图、示意图，准确度仍需人工复核
- API端2K以上分辨率当前是beta；复杂提示词延迟最高可达2分钟
- 重复角色或品牌元素的连续生成偶尔会失稳

---

## 七、安全三层栈

延续1.5的三层结构：上游文本拒绝 → 下游图像/输入双重检查 → 最终输出审查。

**有意思的差异**：Thinking模式从源头产生的违规图本来就少（6.7% vs Instant的22.0%），原因是thinking模型用Safe Completions把对抗prompt转译成安全版本，而不是直接拒绝。

---

## 八、核心洞察

1. **思考模式是最大范式变化**：不是画得更好，是替你想清楚
2. **中文文字渲染突破**：国内用户最直接受益的变化
3. **多尺寸广告一次生成**：营销工作流的颠覆
4. **API定价上涨56-59%**：成本压力显著增加
5. **安全能力与创意能力同步提升**：思考模式自带更好的安全过滤
